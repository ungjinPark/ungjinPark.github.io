시작하기

이 챕터는 기본 사용방법에 대해서 다룹니다.

이 챕터는 다음과 같은 하위 챕터로 구성되어 있습니다.

  • Step1 - 당신의 첫 번째 트리뷰를 생성해보세요: 웹 문서에 대한 트리뷰 스크립트 설치 방법을 설명합니다.
  • Step2 - 마크업 구조: HTML에서 트리뷰 요소를 사용하는 방법을 설명합니다.
  • Step3 - HTML 마크업으로 기본 이벤트, 기본 상태 설정하기: 트리뷰 요소의 초기 기본 상태를 설정하는 방법을 설명합니다.

계속 사용을 읽으려면 Shift + Tab을 눌러 이 페이지 왼쪽에 있는 트리뷰에 액세스한 다음 오른쪽 화살표 키를 눌러 1장 하위 장을 펼칩니다.

마우스 왼쪽 버튼을 클릭하여 트리 항목의 내용 패널을 활성화하고 왼쪽 버튼을 두 번 클릭하여 하위 항목을 확장할 수도 있습니다.

하위 장이 확장된 경우 아래쪽 화살표를 눌러 트리 항목을 탐색한 다음 Enter 키를 눌러 내용 패널을 활성화하고 계속 읽으십시오.

트리 항목을 활성화한 경우, 이 페이지의 오른쪽에 있는 패널이 내용이 변경됩니다.

Treeview.js 패키지의 기본 파일

문서에서 다음 파일 목록의 기본 스크립트 파일을 가져와야 합니다.

  1. treeview.js[main file]: 주요 소스 코드가 있는 파일입니다.
  2. treeview.config.js[running auto-creation of the treeview]:기본 스크립트 파일이 문서의 Treeview.js 패턴을 인식하여 트리 보기 생성을 실행하도록 합니다. 트리 힌트 알림의 메시지 텍스트를 여기서 설정할 수 있습니다.

하위 트리 항목을 가진 요소 전용

이 속성은 트리 항목 오브젝트에 있는 하위 트리를 펼치거나 접습니다.

하위 트리가 확장되기를 원한다면 true, 아니라면 false 값을 설정하세요.

Treeitem.state_expand=boolean (Setter)

TreeItemInstance.state_expand:boolean

이 속성은 부울 값을 반환합니다. aria-expanded 속성의 값이 null이나 'false' 값이라면, false 부울 값을 반환합니다. 그리고 null이나 false가 아니라면 true 부울 값을 반환합니다.
