NVDA란? : NVDA 이모저모

NVDA는 호주에 등록된 비영리 단체인 NVAccess라는 단체의 이름으로 후원을 받으며 여러 개발자가 모여 Python 언어로 기반으로 개발하고 있는 무료 오픈소스 스크린리더로 간편한 설치 구조와 가벼운 것이 특징입니다.

NVDA 스크린리더의 기반을 만든 마이클 커런(Michael Curran)이며, NVAccess에서 제임스 테(James Teh)와 쿠란이 개발에 참여, 관리하고 있습니다.

이런 내용이 안 궁금하실 수 있겠지만, 적어도 자기가 쓰는 스크린 리더가 어떻게 생겨 먹었는지는 알고 쓰셔야 도리?에 맞겠지요. 안 궁금하면 H 키나 q열 위에 위치한 숫자 버튼 중 2 키를 눌러 이 내용을 넘기세요.

NVDA는 Non-Visual Desktop Access의 약자(줄임말)로 직역하자면 “시각정보 없이 데스크탑 접근하기”와 같은 다소 긴 뜻을 가지고 있습니다.

“난.. 노트북인데?”하며 데스크탑에 접근한다고 오해하진 마세요. 가벼운 스크린리더라곤 하나 랩탑 자판도 지원하는 강력한 친구니까요. :)

오픈소스는 코드가 외부에 공개되고 있다는 것으로, Python 언어만 할 줄 안다면 당신도 이 스크린리더의 기능 추가, 버그 수정, 확장 기능을 만들 수 있습니다. Python만 할줄 아신다면 개발에 참여해 보세요.

들어가기 앞서

NVDA의 기본 언어는 영어입니다. NVDA의 변경사항 문서부터 UI에 담긴 텍스트 등 이 기본 언어에서 여러 언어로 번역되며, 한국어 또한 마찬가지입니다. NVDA는 비영리 단체에서 운영하는 무료 스크린 리더로 개발이나 번역 작업이 다소 자발적인 봉사에 가까우며, 이에 의존적입니다. 위에서도 언급하였지만, Python을 다루실 줄 알며, NVDA를 사용한다면, NVDA 발전에 기여해 주세요.

또한, 영어를 잘 하시거나, 영어를 잘 못한다고 하더라도 적절하게 번역기를 사용하여 내용을 번역할 수 있는 사람을 늘 모집하고 있습니다. 현 번역이 마음에 들지 않는다면 지금 당장 도전해 보세요.

NVDA의 설치부터 NVDA의 각종 기능을 다루는 것까지 기초적, 기본적인 사항을 다룹니다.

NVDA 설치하기

우선 NVDA를 사용하려면 당연히 NVDA부터 설치해야겠지요? NVDA를 설치하는 방법은 센스리더가 아닌 내레이터를 기준으로 설명할 것입니다. 센스리더가 없으신 분도 깔 수 있어야 하니까요.

내레이터를 사용하여 웹을 탐색하자

전혀 화면을 볼 수 없거나, 화면을 볼 수 있으나, 듣는 것보다 시간이 오래 걸린다면, NVDA를 깔기 전까지 대체할 스크린리더가 필요할 것입니다. 센스리더를 갖고 계신 분이라면, 센스리더를 사용해도 좋습니다.

이 문서는 새 컴퓨터를 구매하여 아무것도 설치된 것이 없는 Windows 10, 기본 브라우저인 Edge나 Internet Explorer를 기준으로 합니다. 내레이터를 사용하여 웹을 탑색하도록 합니다.

  1. Ctrl + Logo + Enter를 눌러 내레이터를 호출합니다. 내레이터가 열렸다면 한 개의 대화상자가 나오며, 확인을 누르세요.
  2. 내레이터 기본 설정창이 나타납니다. 이 창을 닫으면 내레이터가 종료되므로 Alt+F4로 닫지 마세요. Logo + 아래 방향 화살표 키를 눌러 내레이터 창을 최소화합니다.
  3. Logo 키를 눌러 시작 목록을 열어주세요. 시작 목록이 열렸다면 여러분의 컴퓨터는 검색창에 초점을 보내줍니다.
  4. Internet Explorer 또는 Edge를 입력한 다음, 브라우저 창을 엽니다. 주소 URL 편집창에 https://nvaccess.org를 입력하거나, 아래 링크로 들어가세요. 주소 URL 편집창에 포커스가 가 있지 않다면, Alt+D나 Ctrl+L를 누르고 입력하세요.
  5. 영어 페이지가 나타나지만 걱정하지 마세요. 다운로드 받는 법은 매우 간단하니까요. Tab 키로 Download 링크를 찾은 다음 Enter로 들어가 주세요.
  6. 다운로드 페이지에 들어왔다면, ‘B’ 키를 눌러 다운로드 단추를 찾아 눌러주세요.
  7. Alt + N을 눌러 다운로드 표시창에 접근한 다음, Tab 키로 “저장” 단추를 찾아 눌러 저장합니다.
  8. 다운로드가 완료되었다면, Shift+Tab이나 Alt+R로 실행 버튼을 찾아 누릅니다.

가급적 위의 방법을 숙지하는 것을 추천드리나, 어떠한 이유로 인해 위 과정을 수행할 수 없다면, 이 링크를 통해 다운로드 받으십시오.

NVDA 설치 및 초기 설정

실행 버튼을 눌러 인트로 BGM이 스피커로 출력된다면 정상적으로 실행된 것으로, 기다리시다 보면 영어로 된 약관 멀티라인 편집창이 나옵니다. 무시하셔도 좋습니다. 우린 이런 따분한 것에 관심 없으니까요.

설치 1. 멀티라인 읽기전용 편집창에서 Tab 키를 누르면 “동의” 체크상자가 있습니다. 이 체크상자에 체크하지 않으면 NVDA 설치 버튼을 누를 수 없으므로, Space 키로 체크해 줍니다. 2. 체크상자를 Space 키로 눌러 체크했다면 Tab 키를 눌러보세요. “컴퓨터에 NVDA 설치(I) 버튼이 있을 것입니다. 이 버튼을 눌러 설치를 진행하세요. 3. 설치 대화상자가 한번 더 나타납니다. 이 대화상자에는 Windows 로그온 중 NVDA 실행 설정, NVDA 실행 단축키 및 아이콘 설정 등의 설정이 있으며, 기본적으로 로그온과 실행 단축키는 체크되어 있습니다. 로그온 시 NVDA 실행과 단축키 부분은 바꾸지 않고 설치를 마저 진행하는 것을 추천합니다. 4. 설치가 완료되었다면”NVDA 시작 대화상자"가 나타납니다.

초기설정 1. NVDA 시작 대화상자에는 다음과 같은 한 개의 콤보상자와 세 개의 체크상자로 구성되어있습니다. 1. 키보드 레이아웃 (K) 콤보상자: 데스크탑용 키보드를 사용하는지, 넘버 패드가 없는 노트북(랩탑)이나 87키 배열의 키보드를 사용하는지를 물어봅니다. 이 설정에 따라 특정 기능의 사용 단축키가 달라집니다. 2. CapsLock 키를 NVDA 기능키로 사용 체크상자: CapsLock 키를 Insert, 넘버 패드 Insert 키와 동일하게 NVDA 기능키로 사용할지를 선택하는 확인란입니다. 3. 로그인 후에 NVDA 시작 (A) 체크상자: Windows 10이 켜지고, 바탕화면 화면이 나타나면 NVDA를 켤 것인지 선택하는 확인란입니다. 4. NVDA 시작 시 이 대화상자 표시 체크상자: NVDA가 켜지면 이 시작 대화상자를 표시할지를 결정하는 확인란입니다. 2. 위 네 가지 설정값은 여러분의 컴퓨터 형태, 키보드 종류, 여러분의 컴퓨터 활용 능력이나 스타일에 따라 결정해주세요. 3. 위 네 가지를 전부 확인하고, 자신에 맞게 설정했다면 Enter를 눌러 해당 대화상자를 닫을 수 있습니다. 해당 대화상자는 나중에 설명할 NVDA 메뉴를 열어서 다시 볼 수 있으니 큰 걱정 마시라.

NVDA를 설치하는 조금 더 쉬운 방법 : 보이스위드(VoiceWith)

넘기지 않고 내용을 꾸준히 읽어 오신 분이라면 NVDA가 오픈소스라는 것을 기억하실 겁니다(아마도?). 오픈소스이기 때문에 우리나라에서도 NVDA를 조금 더 쉽게 사용할 수 있게끔 한국인을 위한 추가기능, TTS를 포함한 NVDA 페키지를 경북점자도서관의 웰북(Wellbook) 페이지에서 배포하고 있습니다. 따라서 보이스위드의 플러그인을 제외한 사용법은 기본 NVDA와 동일합니다. 같은 스크린리더니 당연하겠지요?

보이스위드는 NVDA를 기반으로 한 NVDA 스크린리더 페키지로,경북점자도서관에서 사회복지공동모금회의 지원을 받아 개발되었습니다.

보이스위드는 센스리더처럼 일부 기능을 사용하려면 회원인증을 받아야 하는 점이 있지만, 다운로드 자체는 회원가입 없이 가능합니다. https://nvda.or.kr/voicewith/에 접속하여 Tab 키로 “보이스위드 다운로드” 링크를 찾아 눌러서 내려 받으실 수 있습니다. 보이스위드의 설치는 설치 파일로 되어 있으며, 음성으로 안내되기 때문에 따로 다루지는 않습니다.

작성자 추천 초기설정값

  1. 키보드 레이아웃: 넘버패드가 없다면 laptop, 넘버패드가 있다면 desktop으로 설정하세요.
  2. CapsLock 키를 NVDA 기능키로 사용: 개인적으로 추천합니다. 오른손 자리의 키가 포함된 기능키를 쓸 때 유용합니다. 기존의 Capslock 키는 두번 누르는 것으로 대체됩니다.
  3. 로그인 후에 NVDA 시작 (A): 화면을 보기 어려운 시각장애인 사용자라면 반드시 체크해 주세요. 센스리더나 JAWS와 같은 다른 스크린리더를 같이 사용하신다면, 서로 겹쳐서 실행되지 않도록 잘 선택해 주세요.
  4. NVDA 시작 시 이 대화상자 표시: 체크를 해제해 주세요. 그렇지 않다면 여러분이 컴퓨터를 켜서 NVDA가 켜지거나, 여러분이 NVDA를 껏다 킬 때 마다 귀찮게 이 시작 대화상자가 나타날 겁니다.

보이스위드, 이런분에게 추천합니다

  • NVDA를 처음 사용하시는 분들께 추천합니다.
  • 보이스위드는 타자 연습, 독서기 플러그인을 지원합니다. NVDA로 컴퓨터를 처음 배우시는 컴퓨터 초심자 분들께 추천합니다.
  • 보이스위드는 Selvy TTS를 기본으로 제공합니다. TTS를 받는 것에 익숙하지 않고, 유료 TTS를 쓰기 부담스러운 분들께 추천합니다.
  • 기본 NVDA는 한글과컴퓨터의 한글 오피스를 지원하지 않습니다. 보이스위드는 한글과컴퓨터 오피스 지원 플러그인을 제공합니다. HWP 문서를 자주 사용하시는 분들께 추천드립니다.
  • 카카오톡을 NVDA로 사용하시고자 하는 분들께 추천드립니다.

보이스위드에 대한 주의사항

  • (처음 사용자) 타 스크린리더 사용 시 반드시 보이스위드를 받은 다음 아래를 참고하여 환경 설정을 해주셔야 합니다.
  • (기존사용자) 이미 설치된 NVDA가 있다면 기존에 사용하던 추가기능이 사라지므로, 반드시 휴대용 버전 만들기로 백업해주세요.
  • (저시력 기존 사용자) 카카오톡이나 한글 등, 기존 NVDA에서 완전히 지원하지 않던 기능을 보이스 위드를 통해 지원받을 수 있으나, 해당 프로그램 사용 도중 NVDA를 껐다 키는 동작을 하게 되면 프로그램 멈춤 또는 오작동이 발생할 수 있습니다.

NVDA 시작

NVDA는 Ctrl + Alt + N을 눌러 시작할 수 있습니다. 다만, 주의할 점이라면, NVDA를 설치했을 때 바탕 화면에 추가되는 NVDA 바로가기 아이콘을 제거해서는 안 됩니다. NVDA는 이 바로가기 아이콘에 시작 단축키를 할당해 놓았기 때문입니다. 실수로 제거하였다면, NVDA 설치 파일로 아이콘을 다시 생성할 수 있습니다(기존 NVDA가 설치되어 있다면 재설치해도 설정값이 날라가진 않습니다).

NVDA를 종료하고 싶다면, Insert + Q를 눌러 종료하거나, Insert + N을 눌러 NVDA 메뉴를 열고, 메뉴 끝에 있는 “종료” 항목을 통해 종료할 수 있습니다.

NVDA 메뉴 탐색하기

NVDA의 시작과 종료 단축키를 제외하고, 여러분이 NVDA에서 가장 처음으로 배울 단축키 중 하나는 NVDA+N으로, NVDA 메뉴를 여는 단축키입니다.

이제부터 이 문서에서 단축키를 설명할 때 말하는 NVDA 키는 insert, numpad insert 또는 Capslock 키를 의미합니다. NVDA+N을 예로 들자면 insert+N, CAPSLOCK을 사용하도록 하였다면 CAPSLOCK+N 입니다. 센스리더를 사용해 보셨던 분들이라면 CTRL + \ (백슬래시)로 센스리더 설정 창을 열던 것과 비슷한 개념이라고 보시면 되겠습니다.

이 단축키를 누르면 NVDA 메뉴가 팝업 메뉴로 나타납니다. 위/아래 화살표 키로 탐색할 수 있으며, ESC키로 닫을 수 있습니다. “하위메뉴”라고 읽어주는 항목은 오른쪽 화살표 키로 하위 메뉴를 열 수 있으며, 마찬가지로 위/아래 화살표 키로 탐색할 수 있습니다. 모든 항목은 Enter 키로 확장하거나, 메뉴를 활성화할 수 있습니다.

메뉴 구성

  1. 설정 (P): NVDA의 각종 설정 메뉴가 있는 항목입니다. 하위 메뉴는 다음과 같습니다.
  2. NVDA 설정 (S): NVDA 설정창을 엽니다. 기본적으로 “일반 설정” 항목이 활성화되어 있습니다.
  3. 음성 발음 사전 (D): 음성 엔진이 부정확하게 읽거나 읽는 방법이 다를 때 단어의 발음, 글자를 교정하는 설정 창을 엽니다.
  4. 구두점/기호 발음 설정 (P): 기호, 구두점과 같은 특수한 문자를 무엇이라고 읽을지 설정하는 창을 엽니다.
  5. 제스처 설정 (N): NVDA의 키보드 단축키, 터치 스크린 동작을 보고, 수정할 수 있는 트리뷰 형태의 창을 엽니다.
  6. 도구: 일반적인 사용자, 개발자에게 필요한 NVDA 도구를 모아둔 하위 메뉴가 있는 항목입니다. 하위메뉴는 다음과 같습니다.
  7. 로그 보기 (V): 개발자용입니다. NVDA의 오류를 잡거나 기능을 테스트할 때 쓰는 디버깅용 로그입니다. 일반적으로는 쓸 일이 없습니다.
  8. 음성 출력 뷰어(S): NVDA 음성이 말한 내용을 표시하는 창을 엽니다. Alt+Tab으로 접근 가능합니다.
  9. 점자 뷰어: NVDA 음성이 말한 내용을 점자 유니코드로 표시하는 창을 엽니다. Alt+Tab으로 접근 가능합니다.
  10. 파이썬 콘솔: NVDA 제어 모듈을 포함한 파이썬 내장 콘솔을 엽니다. 개발자용이지만 파이썬 기초를 배울 때 도움을 주는 의외의 기능을 하기도 합니다.
  11. 추가 기능 관리 (A): NVDA에 기본적으로 없는 기능을 추가하거나, 설치된 추가 기능을 제거, 비활성화하는 등 관리를 하는 창을 엽니다.
  12. 휴대용 버전 만들기 (C): NVDA의 휴대용 버전을 만들어주는 창을 엽니다. 유저가 선호하는 값으로 설정한 NVDA를 그대로 휴대용 버전으로 만들어 어디서나 설치 없이 사용할 수 있으며, 설치도 가능합니다.
  13. COM Registration 재설정: 응용 프로그램을 못 읽는 상황에 시스템 레지스트리를 수정하여 문제를 해결할 때 사용합니다. 하지만, 쓸 일이 거의 없습니다.
  14. 플러그인 새로고침: NVDA를 끄지 않고, 설치된 추가 기능만을 다시 불러옵니다. 특정 플러그인은 다시 불러와졌을 때, 중복된 메뉴가 생기는 등의 버그가 있으므로, 추가기능(플러그인)을 받았을 때, NVDA를 끄고 다시 켜는 것을 추천합니다.
  15. 도움말 (H): NVDA의 각종 설명서, 정보 창 등을 열 수 있는 하위 항목이 있는 메뉴 입니다.
  16. 사용 설명서 (U): NVDA의 공식 사용 설명서 웹문서를 기본 브라우저로 엽니다.
  17. NVDA 기능키 목록 (Q): NVDA의 기능키를 모아둔 웹문서를 기본 브라우저로 엽니다.
  18. NVDA 변경 이력 (N): NVDA의 버전별 변경사항을 볼 수 있는 웹문서를 기본 브라우저로 엽니다.
  19. NVDA 웹 사이트 (W): NVAccess 웹페이지를 기본 브라우저로 엽니다.
  20. 사용 약관 (I): NVDA의 약관 및 라이선스 정보가 담긴 텍스트 파일을 엽니다.
  21. NVDA 개발 공헌자 목록 (O): NVDA 개발 및 발전에 기여한 공헌자 목록이 담긴 텍스트 파일을 엽니다.
  22. NVDA 시작 대화상자 (L): NVDA를 처음 설치했을 때 표시되는 초기 설정 창을 엽니다.
  23. 업데이트 확인 (C): NVAccess에서 공개한 새로운 NVDA 업데이트가 있는지 확인하고, 업데이트를 진행할 수 있는 창을 엽니다.
  24. NVDA 정보 (A): 현재 설치된 NVDA의 버전 정보를 볼 수 있는 창을 엽니다.
  25. 사용자 프로필 (C): 여러 사람이 NVDA를 함께 쓸 때, 선호하는 설정값을 프로필로 저장하고, 토글할 수 있는 창을 엽니다.
  26. 저장된 환경설정으로 되돌리기 (R): NVDA의 설정값이 의도와 맞지 않게 설정되었을 때, 저장된 환경설정으로 되돌립니다.
  27. 기본 환경설정으로 초기화 (R): NVDA가 사용자의 의도와 맞지 않게 동작할 때, 모든 설정을 초기화한 NVDA로 되돌립니다.
  28. 환경설정 저장 (S): 현재 환경설정을 기억합니다.
  29. 후원 (D): NVDA를 관리하는 NVAccess에 후원하는 웹페이지로 연결됩니다.
  30. 종료 (X): NVDA를 끄거나 다시 시작하는 등의 옵션이 있는 NVDA 종료 대화상자를 엽니다.

NVDA+N으로 여는 NVDA 메뉴의 기본적인 항목을 모두 살펴 보았습니다. 지금은 ‘저런 것들이 있구나’ 정도로 생각하고 이해할 필요 없이 넘어가셔도 좋습니다. 배우지 않은 내용이 더 많을테니까요.

NVDA 설정하기

드디어 NVDA를 설정하는 방법에 대해 설명합니다. NVDA를 설정하는 방법은 여러 가지가 있으나, 가장 기본은 위에서 봤던 NVDA 메뉴-> 설정(P) -> NVDA 설정(S) 으로 NVDA 설정창에서 접근하는 겁니다.

NVDA 설정창을 열면 크게 다음과 같은 두 영역으로 창 화면이 나뉩니다. 1. 설정 패널(C) 목록: 무엇을 설정할 것인지 선택하는 목록입니다. 위 또는 아래 화살표 키로 목록에서 한 개의 항목을 선택하여 설정들을 볼 수 있습니다. Home 또는 End 키로 첫 항목과 끝 항목으로 이동할 수 있습니다. 2. 설정 속성 페이지: 설정 패널(C) 목록에서 선택한 항목에 따라 표시되는 설정들이 바뀌는 영역입니다. 설정 패널 목록에서 “일반 설정”을 방향키로 선택했다면 이 영역에는 일반 설정과 관련된 설정이 모여있는 “일반 설정, 속성 페이지”가 표시됩니다.

그런데, 우리가 어떤 민족입니까? 빠른 걸 최우선으로 생각하는 민족입니다. 설정 메뉴에서 NVDA 설정을 찾아 누르는 것이 매우 귀찮습니다. 언제, 어디서나 컴퓨터가 작업을 수행할 수 없는 상황이 아니라면 우리는 CTRL+NVDA+G 를 사용하여 NVDA의 설정 을 열 수 있습니다.

다만, NVDA 메뉴를 통해 접근하는 것과 차이가 있습니다. NVDA 메뉴-> 설정(P) -> NVDA 설정(S) 으로 설정창에 접근했을 때는 설정 패널(C) 목록을 방향키로 선택할 수 있는 상황입니다. 하지만, 위와 같은 단축키로 접근했을 때는 해당 설정의 첫 번째 구성요소에 초점이 제공됩니다. NVDA 메뉴에서 접근했던 것 처럼, 다른 설정으로 가고싶다면, Shift+Tab으로 설정 패널(C) 목록 상자에 접근하여야 합니다.

(키보드 접근 기초 팁) 아까부터 여러 곳에 괄호 안에 알파뱃이 써져있는데 이건 무엇인가요

Windows는 오래전부터 버튼이나, 체크상자, 라디오 버튼, 선택 가능한 목록 상자 등을 빠르게 키보드 접근이 가능하도록 바로가기 단축키를 제공했습니다. 괄호에 알파뱃이 써져있다면, 그것은 그 곳으로 빠르게 이동하는 단축키입니다. NVDA 메뉴와 같은 메뉴에서는 괄호에 써져 있는 알파뱃 키를 누르면 해당 메뉴로 빠르게 이동할 수 있습니다.

예를 들어, NVDA+N으로 NVDA 메뉴를 열고, 도움말 메뉴로 이동하려면, 두 개의 메뉴를 지나가야 합니다. 그러나 도움말(H)에서 괄호에 H가 써져있으므로, H를 누르면 메뉴를 건너뛰고 도움말 메뉴를 탐색해 줍니다.

반면, NVDA 메뉴나, 여러분이 팝업 키를 눌러 나오는 메뉴 구성요소와 다르게, 대화상자와 같은 창 안에 있는 구성요소에 이 괄호가 써져 있다면 Alt 키를 같이 눌러주어야 합니다. 예를 들어, 여러분이 흔히 볼 수 있는 “예” 또는 “아니오”를 선택하는 대화상자의 이 두 버튼에는 “예”는 Y, “아니오”는 N이 괄호에 써져있습니다. Tab 키로 이 버튼을 찾아 누를 수도 있지만, Alt+Y, Alt+N으로도 이 버튼을 누를 수 있습니다.

(컴퓨터 기초 용어)그런데, 초점이 뭐예요?

여러분도 익히 아는 내용이지만, 컴퓨터를 조작하는 방법에는 두 가지가 있습니다. 마우스로 포인터를 이동하여 클릭하는 방법과 여러분처럼 키보드 조작, 키보드 단축키만으로 컴퓨터를 사용하는 방법이 있습니다.

키보드를 사용하여 컴퓨터를 탐색 할 때, 가장 많이 사용하는 키는 Tab 키로, 버튼이나, 링크, 체크 상자, 라디오 버튼, 콤보 상자, 편집창 등, 누르거나, 방향키로 선택하거나, 무엇을 입력할 수 있는 구성 요소들을 탐색할 때, 이 Tab 키를 사용합니다.

이제부터 위에서 설명한 구성 요소들을 “상호작용 가능한 요소”라고 표현하겠습니다. 상호작용 가능한 요소는 모두 Tab 키로 접근 가능합니다. 이 Tab 키로 상호작용 가능한 요소에 접근하는 것을 “초점을 보낸다”라고 합니다. 다른 표현으로는 영어식으로 “포커스(주시)”라고도 합니다.

여러분이 Tab 키로 “예(Y)” 버튼을 찾았다면, 초점이 “예(Y), 버튼”에 있는 것이라고 표현할 수 있습니다.

NVDA 설정의 모든 설정값은 상호작용 가능한 요소이므로, 단순히 Tab 키를 누르는 것만으로 탐색이 가능합니다.

(Windows 기초) 무언가를 설정하는 대화상자에서는 Enter 키를 신중히 눌러주세요

NVDA 설정, NVDA 시작 대화상자 등, 무언가 체크할 수 있거나, 선택할 수 있는 설정창은 기본적으로 Enter 키에 기본값으로 취소나 확인 버튼 등을 누르는 동작을 수행합니다. 예를 들어, 체크 상자를 선택하고 싶을 때 Enter를 누르면, 설정이 저장되며 나가집니다. 설정에서 누르기 동작은 “Enter” 키가 아닌 “Space Bar”키를 사용해 주세요.

일반 설정

자, 설정창의 기본적인 설명이 끝났습니다. 지금부터는 설정 패널에 있는 설정들을 하나씩 살펴봅시다. 첫 번째는 일반 설정입니다. NVDA 메뉴에서 NVDA 설정을 열었을 때, 기본으로 선택되어 표시되는 설정들입니다.

일반 설정에서 설정할 수 있는 것들은 다음과 같습니다.

  1. NVDA 사용 언어(다시 시작해야 함) (L): 말 그대로, NVDA의 언어를 설정합니다. 대강이라도 알고 있는 언어가 아니라면 바꾸는 것을 추천하지 않습니다.
  2. 종료 시 NVDA 환경 설정저장(S): 기본값은 “선택됨”이며, NVDA가 종료될 때, 설정값을 저장합니다. 선호하지 않는다면 선택을 해제하고, 수동으로 저장해도 무방합니다.
  3. 종료 시 NVDA 종료 옵션 표시(W): 기본값은 “선택됨”이며, NVDA를 종료할 때, 다시 시작, 종료 등의 콤보상자가 있는 대화상자를 표시할 것인지에 대한 설정입니다. 바로 종료되게끔 하고싶다면 선택을 해제하세요.
  4. NVDA 시잣 및 종료음 재생 (P): 기본값은 “선택됨”이며, NVDA를 시작하거나, 종료할 때, 멜로디 톤이 재생됩니다.
  5. 로깅 레벨(O): 일반적으로 사용하지 않는 설정입니다. NVDA의 기능 개발 및 버그 수정 시에만 설정값을 바꿉니다.
  6. 로그인 후에 NVDA 시작(A): NVDA 시작 대화상자에서 있던 설정값으로, 해당 대화상자에서 설정했던 값으로 설정되어 있습니다. NVDA를 보이스위드로 받았고, NVDA보다 센스리더를 주로 사용하는 분이라면, 이 체크상자를 확인하고, 시작 시 실행되지 않도록 선택을 해제해 주세요.
  7. 로그인 중에 NVDA 사용: 컴퓨터가 처음 켜져 로그오프 상태일 때, NVDA가 화면을 읽어주게 할 것인지를 선택합니다.
  8. Windows 로그인 및 보안 화면에서 현재 저장된 설정 사용 (관리자 권한 필요): 이 설정 버튼은 로그인 중에 NVDA 사용() 을 위한 하위 설정으로, Windows 로그인 화면에서 NVDA의 설정값, 음성 엔진을 적용하려면 이 버튼을 눌러야 합니다. 만약 이 설정을 하지 않고, 로그인 중에 NVDA 사용 만을 체크한다면, 로그인 화면에서 Windows 기본 음성이 동작합니다.
  9. NVDA에서 업데이트를 자동으로 확인 (U): NVDA가 실행되어 있을 때, 버전을 체크하여, 최신 버전이 있다면 대화상자로 알립니다. 기본값으로 선택되어 있습니다.
  10. NVDA 실행 시 설치 준비중인 업데이트 알림 (P): NVDA 시작 시, 설치 준비중인 업데이트 내용을 안내합니다. 기본값으로 선택되어 있습니다.
  11. NVDA 프로잭트에 NVDA 사용 통계를 전송하도록 허용: NVAccess의 NVDA 프로젝트(개발팀)에 사용 통계를 전송합니다. 전송된 정보는 NVDA 사용성 및 문제 개선에 사용됩니다.

음성 출력

일반 설정에 대해 모두 알아보았습니다. 일반 설정은 NVDA의 기본적인 환경을 모아놓은 것이라고 할 수 있습니다.

음성 출력 설정값에 대해 설명하기 앞서 NVDA의 기능 조합키는 영어단어의 첫 글자 이니셜을 딴 것들이 많습니다.

음성 출력 설정은 CTRL+NVDA+V로 Voice(음성)의 첫 글자 이니셜을 따른 것입니다.

마찬가지로, 이전에 설명한 일반 설정에 들어가는 조합키인 CTRL+NVDA+G는 General(일반)의 앞 글자 이니셜입니다.

NVDA는 영어 단어와 친숙해진다면 이러한 단축키를 외우는 데에 도움이 됩니다.

다음은 음성 출력에서 설정할 수 있는 것들입니다.

  1. 움성 엔진 (S): 음성 엔진의 종류를 설정합니다. 나중에 설명할 추가 기능을 통해 설치한 음성 엔진, Windows에 기본적으로 설치된 음성 엔진 종류를 설정할 수 있습니다. 현재 선택된 음성 엔진을 읽기전용 편집창에서 볼 수 있으며, 편집창 다음에 있는 변경(C) 버튼을 눌러 “음성 엔진 선택” 대화상자를 열 수 있습니다. 음성엔진 선택 대화상자는 다음 세 가지 값을 설정할 수 있습니다.
  2. 음성 엔진 (S): 음성 엔진의 종류를 선택합니다. 음성 엔진은 목소리의 종류를 얘기하는 것이 아니며, 여러 음성을 모아둔 프로그렘을 의미합니다. 콤보상자로 목록이 제공되며, 당연한 얘기지만 설치된 음성 엔진의 수에 따라 사람마다 항목 갯수가 다릅니다.
  3. 음성 출력 장치 (D): 음성이 어떤 장치에서 나오게 할 것인지 선택합니다. 예를들어, 오디오 장치를 두 종류 사용하고 있다면, 한 곳에서는 NVDA 음성만 출력되게 할 수 있습니다.
  4. 오디오 도킹 모드 (D): 소리가 나는 미디어 콘텐츠를 보며 스크린리더를 사용할 때, 멀티미디어의 음량을 줄이고, 스크린리더의 음성을 부각시킬 것인지, 같은 음량으로 나오게 할 것인지 등을 선택할 수 있습니다.

(Windows 기초) 동일한 바로가기 단축키가 여러 곳에 있으면 어떻게 되나요?

음성 엔진 선택 대회상자 를 보면 바로가기 키가 D로 되어 있는 설정 항목 두 개, 음성 출력 장치 (D) 오디오 도킹 모드 (D) 가 보입니다.

이럴 때는 해당 단축키를 한 번 더 누르면 다음 항목으로 이동합니다. 처음 alt+d를 누르면 바로가기 키가 D로 지정된 첫 번째 설정 요소로 이동하며, 한 번 더 누르면 바로가기 키가 D로 지정된 다음 설정 요소로 이동합니다.

두 번째 바로가기 요소에서 첫번째 바로가기 요소로 돌아가고 싶다면, Shift를 누르고 Alt+D를 누르면 바로가기 단축키가 D로 지정된 이전 요소를 탐색합니다. 단축키로 무언가를 탐색 할 때 Shift를 누르는 것은 이전 요소 탐색을 의미하는 것을 기억해두시면 좋습니다.
