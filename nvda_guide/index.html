NVDA란? : NVDA 이모저모

NVDA는 호주에 등록된 비영리 단체인 NVAccess라는 단체의 이름으로 후원을 받으며 여러 개발자가 모여 Python 언어로 기반으로 개발하고 있는 무료 오픈소스 스크린리더로 간편한 설치 구조와 가벼운 것이 특징입니다.

NVDA 스크린리더의 기반을 만든 마이클 커런(Michael Curran)이며, NVAccess에서 제임스 테(James Teh)와 쿠란이 개발에 참여, 관리하고 있습니다.

이런 내용이 안 궁금하실 수 있겠지만, 적어도 자기가 쓰는 스크린 리더가 어떻게 생겨 먹었는지는 알고 쓰셔야 도리?에 맞겠지요. 안 궁금하면 H 키나 q열 위에 위치한 숫자 버튼 중 2 키를 눌러 이 내용을 넘기세요.

NVDA는 Non-Visual Desktop Access의 약자(줄임말)로 직역하자면 “시각정보 없이 데스크탑 접근하기”와 같은 다소 긴 뜻을 가지고 있습니다.

“난.. 노트북인데?”하며 데스크탑에 접근한다고 오해하진 마세요. 가벼운 스크린리더라곤 하나 랩탑 자판도 지원하는 강력한 친구니까요. :)

오픈소스는 코드가 외부에 공개되고 있다는 것으로, Python 언어만 할 줄 안다면 당신도 이 스크린리더의 기능 추가, 버그 수정, 확장 기능을 만들 수 있습니다. Python만 할줄 아신다면 개발에 참여해 보세요.

들어가기 앞서

NVDA의 기본 언어는 영어입니다. NVDA의 변경사항 문서부터 UI에 담긴 텍스트 등 이 기본 언어에서 여러 언어로 번역되며, 한국어 또한 마찬가지입니다. NVDA는 비영리 단체에서 운영하는 무료 스크린 리더로 개발이나 번역 작업이 다소 자발적인 봉사에 가까우며, 이에 의존적입니다. 위에서도 언급하였지만, Python을 다루실 줄 알며, NVDA를 사용한다면, NVDA 발전에 기여해 주세요.

또한, 영어를 잘 하시거나, 영어를 잘 못한다고 하더라도 적절하게 번역기를 사용하여 내용을 번역할 수 있는 사람을 늘 모집하고 있습니다. 현 번역이 마음에 들지 않는다면 지금 당장 도전해 보세요.

NVDA 설치하기

우선 NVDA를 사용하려면 당연히 NVDA부터 설치해야겠지요? NVDA를 설치하는 방법은 센스리더가 아닌 내레이터를 기준으로 설명할 것입니다. 센스리더가 없으신 분도 깔 수 있어야 하니까요.

내레이터를 사용하여 웹을 탐색하자

전혀 화면을 볼 수 없거나, 화면을 볼 수 있으나, 듣는 것보다 시간이 오래 걸린다면, NVDA를 깔기 전까지 대체할 스크린리더가 필요할 것입니다. 센스리더를 갖고 계신 분이라면, 센스리더를 사용해도 좋습니다.

이 문서는 새 컴퓨터를 구매하여 아무것도 설치된 것이 없는 Windows 10, 기본 브라우저인 Edge나 Internet Explorer를 기준으로 합니다. 내레이터를 사용하여 웹을 탑색하도록 합니다.

  1. Ctrl + Logo + Enter를 눌러 내레이터를 호출합니다. 내레이터가 열렸다면 한 개의 대화상자가 나오며, 확인을 누르세요.
  2. 내레이터 기본 설정창이 나타납니다. 이 창을 닫으면 내레이터가 종료되므로 Alt+F4로 닫지 마세요. Logo + 아래 방향 화살표 키를 눌러 내레이터 창을 최소화합니다.
  3. Logo 키를 눌러 시작 목록을 열어주세요. 시작 목록이 열렸다면 여러분의 컴퓨터는 검색창에 초점을 보내줍니다.
  4. Internet Explorer 또는 Edge를 입력한 다음, 브라우저 창을 엽니다. 주소 URL 편집창에 https://nvaccess.org를 입력하거나, 아래 링크로 들어가세요. 주소 URL 편집창에 포커스가 가 있지 않다면, Alt+D나 Ctrl+L를 누르고 입력하세요.
  5. 영어 페이지가 나타나지만 걱정하지 마세요. 다운로드 받는 법은 매우 간단하니까요. Tab 키로 Download 링크를 찾은 다음 Enter로 들어가 주세요.
  6. 다운로드 페이지에 들어왔다면, ‘B’ 키를 눌러 다운로드 단추를 찾아 눌러주세요.
  7. Alt + N을 눌러 다운로드 표시창에 접근한 다음, Tab 키로 “저장” 단추를 찾아 눌러 저장합니다.
  8. 다운로드가 완료되었다면, Shift+Tab이나 Alt+R로 실행 버튼을 찾아 누릅니다.

가급적 위의 방법을 숙지하는 것을 추천드리나, 어떠한 이유로 인해 위 과정을 수행할 수 없다면, 이 링크를 통해 다운로드 받으십시오.

NVDA 설치 및 초기 설정

실행 버튼을 눌러 인트로 BGM이 스피커로 출력된다면 정상적으로 실행된 것으로, 기다리시다 보면 영어로 된 약관 멀티라인 편집창이 나옵니다. 무시하셔도 좋습니다. 우린 이런 따분한 것에 관심 없으니까요.

설치 1. 멀티라인 읽기전용 편집창에서 Tab 키를 누르면 “동의” 체크상자가 있습니다. 이 체크상자에 체크하지 않으면 NVDA 설치 버튼을 누를 수 없으므로, Space 키로 체크해 줍니다. 2. 체크상자를 Space 키로 눌러 체크했다면 Tab 키를 눌러보세요. “컴퓨터에 NVDA 설치(I) 버튼이 있을 것입니다. 이 버튼을 눌러 설치를 진행하세요. 3. 설치 대화상자가 한번 더 나타납니다. 이 대화상자에는 Windows 로그온 중 NVDA 실행 설정, NVDA 실행 단축키 및 아이콘 설정 등의 설정이 있으며, 기본적으로 로그온과 실행 단축키는 체크되어 있습니다. 로그온 시 NVDA 실행과 단축키 부분은 바꾸지 않고 설치를 마저 진행하는 것을 추천합니다. 4. 설치가 완료되었다면”NVDA 시작 대화상자"가 나타납니다.

초기설정 1. NVDA 시작 대화상자에는 다음과 같은 한 개의 콤보상자와 세 개의 체크상자로 구성되어있습니다. 1. 키보드 레이아웃 (K) 콤보상자: 데스크탑용 키보드를 사용하는지, 넘버 패드가 없는 노트북(랩탑)이나 87키 배열의 키보드를 사용하는지를 물어봅니다. 이 설정에 따라 특정 기능의 사용 단축키가 달라집니다. 2. CapsLock 키를 NVDA 기능키로 사용 체크상자: CapsLock 키를 Insert, 넘버 패드 Insert 키와 동일하게 NVDA 기능키로 사용할지를 선택하는 확인란입니다. 3. 로그인 후에 NVDA 시작 (A) 체크상자: Windows 10이 켜지고, 바탕화면 화면이 나타나면 NVDA를 켤 것인지 선택하는 확인란입니다. 4. NVDA 시작 시 이 대화상자 표시 체크상자: NVDA가 켜지면 이 시작 대화상자를 표시할지를 결정하는 확인란입니다. 2. 위 네 가지 설정값은 여러분의 컴퓨터 형태, 키보드 종류, 여러분의 컴퓨터 활용 능력이나 스타일에 따라 결정해주세요. 3. 위 네 가지를 전부 확인하고, 자신에 맞게 설정했다면 Enter를 눌러 해당 대화상자를 닫을 수 있습니다. 해당 대화상자는 나중에 설명할 NVDA 메뉴를 열어서 다시 볼 수 있으니 큰 걱정 마시라.

작성자 추천 초기설정값
  1. 키보드 레이아웃: 넘버패드가 없다면 laptop, 넘버패드가 있다면 desktop
  2. CapsLock 키를 NVDA 기능키로 사용: 개인적으로 추천합니다. 오른손 자리의 키가 포함된 기능키를 쓸 때 유용합니다. 기존의 Capslock 키는 두번 누르는 것으로 대체됩니다.
  3. 로그인 후에 NVDA 시작 (A): 화면을 보기 어려운 시각장애인 사용자라면 반드시 체크해 주세요. 센스리더나 JAWS와 같은 다른 스크린리더를 같이 사용하신다면, 서로 겹쳐서 실행되지 않도록 잘 선택해 주세요.
  4. NVDA 시작 시 이 대화상자 표시: 체크를 해제해 주세요. 그렇지 않다면 여러분이 컴퓨터를 켜서 NVDA가 켜지거나, 여러분이 NVDA를 껏다 킬 때 마다 귀찮게 이 시작 대화상자가 나타날 겁니다.

NVDA를 설치하는 조금 더 쉬운 방법 : 보이스위드(VoiceWith)

넘기지 않고 내용을 꾸준히 읽어 오신 분이라면 NVDA가 오픈소스라는 것을 기억하실 겁니다(아마도?). 오픈소스이기 때문에 우리나라에서도 NVDA를 조금 더 쉽게 사용할 수 있게끔 한국인을 위한 추가기능, TTS를 포함한 NVDA 페키지를 경북점자도서관의 웰북(Wellbook) 페이지에서 배포하고 있습니다. 따라서 보이스위드의 플러그인을 제외한 사용법은 기본 NVDA와 동일합니다. 같은 스크린리더니 당연하겠지요?

보이스위드는 NVDA를 기반으로 한 NVDA 스크린리더 페키지로,경북점자도서관에서 사회복지공동모금회의 지원을 받아 개발되었습니다.

보이스위드는 센스리더처럼 일부 기능을 사용하려면 회원인증을 받아야 하는 점이 있지만, 다운로드 자체는 회원가입 없이 가능합니다. https://nvda.or.kr/voicewith/에 접속하여 Tab 키로 “보이스위드 다운로드” 링크를 찾아 눌러서 내려 받으실 수 있습니다. 보이스위드의 설치는 설치 파일로 되어 있으며, 음성으로 안내되기 때문에 따로 다루지는 않습니다.

보이스위드, 이런분에게 추천합니다.
  • NVDA를 처음 사용하시는 분들께 추천합니다.
  • 한글 오피스를 사용하시는 분들께 추천드립니다.
  • 카카오톡을 NVDA로 사용하시고자 하는 분들께 추천드립니다.
보이스위드에 대한 주의사항
  • (처음 사용자) 타 스크린리더 사용 시 반드시 보이스위드를 받은 다음 초기 설정을 해주셔야 합니다.
  • (기존사용자) 이미 설치된 NVDA가 있다면 기존에 사용하던 추가기능이 사라지므로, 반드시 휴대용 버전 만들기로 백업해주세요.
  • (저시력 기존 사용자) 카카오톡이나 한글 등, 기존 NVDA에서 완전히 지원하지 않던 기능을 보이스 위드를 통해 지원받을 수 있으나, 해당 프로그램 사용 도중 NVDA를 껐다 키는 동작을 하게 되면 프로그램 멈춤 또는 오작동이 발생할 수 있습니다.

NVDA 메뉴 탐색하기

여러분이 NVDA에서 가장 처음으로 배울 단축키 중 하나는 NVDA+N로, NVDA 메뉴를 여는 단축키입니다. 이제부터 이 문서에서 단축키를 설명할 때 나오는 NVDA 키는 insert, numpad insert 또는 capslock 키를 의미합니다. NVDA+N을 예로 들자면 insert+N, CAPSLOCK을 사용하도록 하였다면 CAPSLOCK+N 입니다. 센스리더를 사용해 보셨던 분들이라면 CTRL + \ (백슬래시)로 센스리더 설정 창을 열던 것과 비슷한 개념이라고 보시면 되겠습니다.

이 단축키를 누르면 NVDA 메뉴가 팝업 메뉴로 나타납니다. 위/아래 화살표 키로 탐색할 수 있으며, ESC키로 닫을 수 있습니다. “하위메뉴”라고 읽어주는 항목은 오른쪽 화살표 키로 하위 메뉴를 열 수 있으며, 마찬가지로 위/아래 화살표 키로 탐색할 수 있습니다. 모든 항목은 Enter 키로 확장하거나, 메뉴를 활성화할 수 있습니다.
